# Metrics Explained

## Occupancy
- **Achieved Occupancy**: The ratio of active warps on an SM to the maximum number of active warps supported.
- **Theoretical Occupancy**: The limit based on your launch config (block size, shared mem, registers).

## Memory Bandwidth Utilization
Percentage of the GPU's peak memory bandwidth used during the kernel execution.

## Compute Utilization
Percentage of peak TFLOPS used.
